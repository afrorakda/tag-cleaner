# Tag Cleaner
Extract, clean, and deduplicate hashtags instantly from any text.

## Philosophy
* **1 Tool, 1 Action.**
* **No Login / No Ads / No Tracking**
* **Browser-only Privacy** (Works on low-spec devices)

## How to Use
1. **Paste your text** into the input field.
2. **Tap "EXTRACT TAGS"** to collect all unique hashtags.
3. **Tap "COPY"** to grab them and you're ready to post!
